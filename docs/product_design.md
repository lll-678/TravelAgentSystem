# Product Design

## Positioning

Smart Tour Guide is a campus/scenic-area navigation and recommendation demo system for 北京邮电大学沙河校区. It emphasizes map data, graph routing, recommendation explainability, diary/community interaction, food discovery, and admin visibility.

## Target Users

- Visitor: searches places, plans routes, finds nearby facilities, reads diaries.
- Student/staff: uses food recommendation and route planning repeatedly.
- Admin/demo operator: checks data scale, OSM import status, and feature readiness.

## Core User Flows

1. Open home dashboard and inspect recommended destinations.
2. Search destinations or services by keyword.
3. Open map guide and view roads, buildings, and facilities.
4. Plan a route and inspect distance/time/steps.
5. Query nearby facilities by category and graph distance.
6. Browse, publish, rate, and comment on diaries.
7. Recommend foods by cuisine, rating, heat, and distance.
8. Generate mock AIGC diary/storyboard output.
9. Check admin data scale and map import status.

## Demo Boundary

The system is optimized for course demonstration. Authentication, moderation editing, production RBAC, and real external AIGC calls remain future work.
